Examples of my work

I design logos and build websites. Below are some examples of the work I've done. Always adding to the list so stay tuned!


Buzz Clothes wanted an online e-commerce store building and new logo designing. They needed this very quickly to meet a sudden influx of demand so I designed and developed both within 3 days, using Illustrator for the logo and WordPress/WooCommerce for the site.



Ox Designs is a web design and development company. For this design I used WordPress and SiteOrigin's site builder to achieve a modern, fully responsive look. I also created their logo using Adobe Illustrator to ensure the image was fully scalable.



Bluebell Spa wanted a modern responsive website to advertise their spa services. I designed and developed this within a very tight time-frame and budget using WordPress



Cubed Games UK wanted a fully functioning e-commerce store to help sell their products. I designed this to be fully responsive and set up integrated payment systems along with analytics tools so they could track their customer base.

I'm always open to many ways of developing and designing. This website, for example, was built with vanilla Javascript, HTML and CSS but as you can see from the above, I enjoy using WordPress for websites I build for clients. The speed, functionality and usability of WordPress means that costs can be kept down without compromise on quality.
